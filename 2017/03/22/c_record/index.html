原因：答案为10，,9 不要惊慌！ 原因在于计算机会先算后者a++，即第二个数为9
此后a为10，再算第一个数为10，最后输出10,9
你可能要问为什么先算第二个？
原因在于：数据在栈结构中，像弹夹一样，先进后出，压站顺序呢由左往右，所以先算第二个！
另外悄悄告诉你：这个顺序也不是绝度的，是由编译器定的！

C语言二级考试细节

  1. 数组名做为实参传递， 当作数组首地址处理
  2. 数组的首地址是常量，是不可以被赋值的
  3. 指针类型是绝对不可以做加法
  4. c语言中未经初始化的变量的值是不可知的
  5. 若二维数组a有m列,则在a[i][j]前的元素个数为( )
  6. strcmp函数用于比较两个字符串是否相等，若s2与s1一样，则返回值为0，若s1>s1返回大于0的数字
  7. 全局变量未经初始化默认赋值为0
  8. 函数返回值可以和普通值一样，可以出现表达式中，也可以作为函数实参，但不可以作为形参
